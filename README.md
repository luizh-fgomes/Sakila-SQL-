# 📚 Sakila SQL Practice

Este repositório reúne consultas SQL desenvolvidas com o banco de dados **Sakila**, disponibilizado oficialmente para **MySQL**. O objetivo do projeto é praticar e aprimorar conhecimentos em SQL por meio de exercícios e consultas realizadas em um banco de dados relacional utilizado para fins de aprendizado.

## 🎯 Objetivo

Praticar os principais conceitos da linguagem SQL, desenvolvendo consultas que simulam situações comuns do dia a dia de um profissional da área de Banco de Dados.

Ao longo do projeto, serão adicionados exemplos envolvendo:

* SELECT
* WHERE
* ORDER BY
* LIMIT
* INNER JOIN
* LEFT JOIN
* RIGHT JOIN
* GROUP BY
* HAVING
* Funções de agregação (COUNT, SUM, AVG, MAX e MIN)
* Subqueries
* Views
* Outros recursos do SQL conforme o avanço dos estudos.

## 🛠️ Tecnologias Utilizadas

* **MySQL**
* **PopSQL** (utilizado para criação e execução das consultas SQL)
* **Banco de Dados Sakila**

## 📂 Estrutura do Projeto

As consultas serão organizadas por assunto para facilitar a navegação e o acompanhamento da evolução dos estudos.

Exemplo:

```text
01-selects/
02-filtros/
03-joins/
04-group-by/
05-functions/
...
```

## 📖 Sobre o Banco de Dados

O projeto utiliza o **Sakila**, um banco de dados de exemplo desenvolvido e disponibilizado pela equipe do **MySQL**. Ele simula o sistema de uma locadora de filmes e possui diversas tabelas relacionadas, tornando-o ideal para praticar consultas SQL em cenários próximos aos encontrados no mercado.

## 🚀 Objetivo Pessoal

Este repositório faz parte da minha jornada de aprendizado em **Banco de Dados** e será atualizado continuamente conforme estudo novos conceitos e técnicas em SQL.
